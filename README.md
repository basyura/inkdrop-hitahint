# hit a hint

hit a hint for preview mode.

![Screenshot](https://raw.githubusercontent.com/basyura/inkdrop-hitahint/master/images/preview.png)

## Install

```
ipm install hitahint
```

## Keybindings

| Command       | Explanation |
| ------------- | ----------- |
| hitahint:show | show hint   |

## Settings

```json
hintcharacters: {
  title: "hintcharacters",
  type: "string",
  default: "asdfghl",
},
```
